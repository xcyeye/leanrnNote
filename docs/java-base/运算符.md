# 移位运算符

![image-20220727090825543](https://picture.xcye.xyz/image-20220727090825543.png)

```java
int a = 0b00111100;
int b = 0b00001101;

/**
         * 0011 1100
         * 0000 1101
         */
System.out.println(a & b);       // 0000 1100=12  如果相对应位都是1，则结果为1，否则为0
System.out.println(a | b);       // 0011 1101=61  如果相对应位都是 0，则结果为 0，否则为 1
System.out.println(a ^ b);       // 0011 0001=49  如果相对应位值相同，则结果为0，否则为1
System.out.println(~a);          // 1100 0011=-61 按位取反运算符翻转操作数的每一位，即0变成1，1变成0。还需要将正负颠倒
System.out.println(a << 2);      // 0011 1100 左移2位 1111 0000=240, 左移3位 1110 0000=480 按位左移运算符。左操作数按位左移右操作数指定的位数。
System.out.println(a << 3);

System.out.println(a >> 2);      // 0011 1100 右移2位 0000 1111=15, 左移3位 0000 1110=7 按位右移运算符。左操作数按位右移右操作数指定的位数。
System.out.println(a >> 3);

System.out.println(a >>> 2);      // 0011 1100 右移2位 0000 1111=15, 左移3位 0000 1110=7 按位右移补零操作符。左操作数的值按右操作数指定的位数右移，移动得到的空位以零填充。
// >>> 其实和>>差不多，但是并没有<<<这个操作符
```
